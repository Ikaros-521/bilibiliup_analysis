# bilibiliup_analysis

这是一套针对b站up主（尤其是主播）的数据分析系统。主要功能为：**抓取航海榜名单、监控抓取新增粉丝；根据uid文件批量抓取粉丝信息**。

## 抓取航海榜名单

有两个版本，python版本由gpt从js版本重构而来，具有输入对话框，理论上使用更舒适。

对于小白用户：请下载抓取航海榜.exe，可执行文件版本

使用中会请你先输入房间号和主播uid，分别在主播直播间和个人空间的url链接里可以找到
